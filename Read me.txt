Данная программа позволяет ползователю подобрать никнейм для различных
аккаунтов. Никнейм подбирается рандомно.

БД реализована в программе Microsoft SQL Server 2014 Management Studio.
CharacterName -> БД с именами персонажей.

Загрузка БД в программу:
Базы данных (правая копка мыши) -> восстановить бд -> Устройство ->
-> тип носителя резервной копии: файл -> Добавить -> CharacterName.bak
